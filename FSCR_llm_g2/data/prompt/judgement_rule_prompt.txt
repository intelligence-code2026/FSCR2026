# 任务：法律案例相似性分析与裁判规则生成

## 输入变量
- {query}: 查询案例摘要（事实描述+法律争议）  
- {candidate}: 候选案例摘要  
- {reason}: 候选案例的裁判理由  

## 处理步骤
1. **法律相似性判断**  
   - {query}和{candidate}的法律相似性对比维度：  
     ✓ 行为主体类型  
     ✓ 行为客体类型  
     ✓ 行为特征
   - 输出{query}和{candidate}的法律相似性对比结论：`高度相似`/`部分相似`/`不相似`
   

2. **裁判理由影响分析**（仅当高度相似时执行）  
   - 检查`{reason}`中是否明确引用`{candidate}`  
   - 分析影响类型：  
     ✓ 直接引用（作为判例依据）  
     ✓ 实质影响（法律逻辑或标准参考）  
     ✓ 无实质关联 
    

3. **裁判规则生成**  
   - 包含内容：
     规则：[简明规则表述]  
     依据：[相关法条/司法解释]  
     参考案例：[候选案例名称/案号]  
     适用理由：裁判规则对{query}的适用理由

## 输出要求
输出包含以下三项内容：
1{query}和{candidate}的法律相似性对比结论、2裁判理由影响分析结果、3裁判规则生成结果和适用理由。
以上三项用,分开。
