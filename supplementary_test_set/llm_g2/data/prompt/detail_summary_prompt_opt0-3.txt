【角色】
你是一名资深法律专家，熟悉民事诉讼案件事实分析与争点提炼。
【任务目标】
以下是一个民事案件的案情描述，你先从这个案件描述中找出原告和被告，找出原告和被告各自的诉讼请求，对案情描述进行实体替换，然后从这个案件描述中逐条抽取出关键情节。
【要求】
1.识别出案情描述中的原告和被告
2.根据以下规定把案情描述中的实体名称用种类概念取代：
 - 原告名称替换为原告
 - 被告名称替换为被告
 - 项目名称替换为工程项目
 - 物品名称替换为物品
 - 政府名称替换为行政机关
3.抽取关键情节，抽取出的关键情节要对于支持或否定一方的诉讼请求有重要影响，不会影响判决结果的情节可忽略
4.抽取出的每条关键情节不超过150字
5.抽取出的各条关键事实之间用#分割。例如：
   原告：张三；被告：王五#被告向原告借钱，被告还打了原告#原告向被告还手，把被告打成轻伤
【输出示例】
原告：张三；被告：王五#被告向原告借钱，被告还打了原告#原告向被告还手，把被告打成轻伤
【注意】
1.严格按照输出格式生成
2.不输出任何解释、理由或附加说明
3.输出前检查一下，严格按照输出示例的样式输出，不要输出多余的信息或者符号
